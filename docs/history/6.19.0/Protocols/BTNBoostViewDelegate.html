Discussion

Invoked when a user interacts with a BoostView.

Implement this delegate if you intend to show brand terms or details before a PurchasePath is started. By default, the Button SDK will automatically fetch and start PurchasePaths generated by a BoostView.

@code // Fetch & start the purchasePathRequest once you’ve displayed your UI to your user. [Button.purchasePath fetchWithRequest:purchasePathRequest purchasePathHandler:^(BTNPurchasePath purchasePath, NSError error) { if (purchasePath) { [purchasePath start]; } }]; @endcode
